- (id) asAssembly

Convenience method for casting the factory to an TyphoonAssembly sub-class.

TyphoonBlockComponentFactory allows using a TyphoonAssembly interface to pose in front of the factory, in order to resolve components. This avoids the requirement to use "magic strings" when multiple components with the same class are configured in different ways.

Example:

MyAssemblyType* assembly = [factory asAssembly];
//Use the assembly interface instead of a 'magic string'
AnalyticsService* service = [assembly analyticsService];
