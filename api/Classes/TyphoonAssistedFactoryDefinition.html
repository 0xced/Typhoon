TyphoonAssistedFactoryDefinition Class Reference

Inherits from NSObject
Declared in TyphoonAssistedFactoryDefinition.h

Overview

Used to enumerate over factory method selectors and their associated body blocks

Tasks

  • – factoryMethod:body:

    Define a new factory method with the given selector and associating the given block. The block should return a value. The arguments of the block are the arguments of the factory method in the same order, but the factory itself is prefixed as first argument, so the factory method arguments are the second and following arguments.
  •   countOfFactoryMethods

  • – enumerateFactoryMethods:
